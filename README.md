Social Spark is an Android app that was created to give users basic social recommendations depending on the time of day they entered. With an input field, vertically stacked buttons, and a results display area, the application has a simple and easy-to-use interface. Input is processed and suggestions are produced using Kotlin if/else statements, which also include a reset function to clear inputs and error handling for invalid values. GitHub was used to monitor and save the project's code, and an emulator was used for testing to guarantee proper operation. The student modified and implemented the code based on advice from ChatGPT (OpenAI, 2022) and W3Schools (n.d.).
